# จัดเก็บตัวเลข (Number Tracker)

แอปบันทึกรายรับ-รายจ่าย สำหรับ Windows

## ดาวน์โหลด

ไปที่หน้า [Releases](../../releases) แล้วดาวน์โหลดไฟล์ `จัดเก็บตัวเลข.exe`

## คุณสมบัติ
- บันทึกเงินเข้า / เงินออก / ค่างาน
- จัดกลุ่มรายการ
- ค้นหา & กรองตามประเภท/เดือน
- ส่งออก CSV
- รองรับ Dark/Light theme
- ใช้งาน offline — ไม่ต้องต่อเน็ต
- ข้อมูลเก็บในเครื่อง (LocalStorage)
